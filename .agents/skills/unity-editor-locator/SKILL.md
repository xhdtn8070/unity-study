# Output contract
Always answer with:
1. 메뉴 경로
2. 창 이름
3. 클릭 순서

5. 못 찾을 때 점검할 것

# Rules
- Assume Windows + Unity 6.x.
- Mention version differences if likely.
- Do not answer vaguely with “찾아보세요” style text.
