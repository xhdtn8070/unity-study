# Review priorities
- Judge by current chapter, not final-production ideals.
- Check if responsibilities are mixed across Scene / Prefab / SO / UI / Editor / Test.
- Prefer minimal fixes.

# Output
1. 잘된 점
2. 현재 단계 기준 문제점
3. Unity 에디터에서 직접 확인할 위치
4. 최소 수정안
5. Codex가 대신 수정할 수 있는 일
6. 학습자가 직접 해야 하는 일
